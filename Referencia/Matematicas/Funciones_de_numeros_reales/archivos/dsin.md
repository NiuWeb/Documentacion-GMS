# dsin

devuelve el _seno_ del ángulo dado (en grados).

## Síntaxis

  
```gml  
dsin(val);  
```  

## Argumentos

Argumento|Descripción|  
---|---|  
val|El ángulo al cual obtener el _seno_.|  

## Descripción

Esta función permite obtener el valor del _seno_ de un ángulo determinado.

## Devuelve

Número real.

## Ejemplo

  
```gml  
A = dsin(90); //devuelve 1  
```
