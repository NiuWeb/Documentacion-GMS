# math_get_epsilon

Obtiene el épsilon matemático actual

## Sintaxis

  
```gml  
math_get_epsilon();  
```  

## Argumentos

Ninguno.  

## Descripción

Devuelve el valor del épsilon (precisión de aproximación de cifras decimales) actual.

## Devuelve

Nùmero real.

## Ejemplo

  
```gml  
precision = math_get_epsilon();  
```
